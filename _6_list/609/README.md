# 矩陣相加

## 設計說明：
請撰寫一程式，讓使用者建立兩個2*2的矩陣，其內容為從鍵盤輸入的整數，接著輸出這兩個矩陣的內容以及它們相加的結果。

## 輸入說明

兩個2*2矩陣，皆輸入整數

## 輸出說明

矩陣1的內容
矩陣2的內容
矩陣1及矩陣2相加的結果

## 輸入輸出範例
輸入與輸出會交雜如下，輸出的部份以黑底表示
### 範例輸入的部分
```
Enter matrix 1:
[1, 1]: 3
[1, 2]: 5
[2, 1]: 7
[2, 2]: 5
```
```
Enter matrix 2:
[1, 1]: 6
[1, 2]: 9
[2, 1]: 8
[2, 2]: 3
```
### 範例輸出的部分
```
Matrix 1:
**3 5 **
**7 5 **
Matrix 2:
**6 9 **
**8 3 **
Sum of 2 matrices:
**9 14 **
**15 8 **
```
